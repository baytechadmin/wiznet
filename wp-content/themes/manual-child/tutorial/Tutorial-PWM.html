PWM

This PWM Example program is for control LED brightness accroding to PWM value.

Importing PWM Example

  1. Enter the Import into Compiler button.



Let’s open the main.cpp file in Program Workspace. This program controls LED brighness accroding to PWM value. It controls PWM Output Value using write() , pulsewidth_us() and etc. 

#include "mbed.h"

PwmOut pwm0(D3); //Declare PWM Output D3
PwmOut pwm1(D5); //Declare PWM Output D5

int main(void)
{
    int i;
    while(1){ //Execute forever.
        for(i=1000; i>0; i=i-1) { //Variable integer value 1000 to 0

           pwm0.write(i*0.001); //Wirte pwm value 0 to 1

           pwm1.pulsewidth_us(i*20); //Put pulsewidth value

           wait(0.01);
        }
    }
}





PWMOut Class

For PWM , using PWMOut Class in mbed Library .
