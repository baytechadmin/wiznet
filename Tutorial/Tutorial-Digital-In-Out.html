Digital In

This Digital In Example program is for read Digital Input Pin value. User will control Ditigal Input Pin value with Swtich. And LED turns on or off, depending on the Digital input value.

Importing Digital In

This is the Digial In Example program. Click the Import into Compiler button





Let’s open the main.cpp file in Program Workspace. This program reads Digital Input Value from D10 pin and controls LED1 . 
  #include "mbed.h"

  DigitalOut myled_R(LED1); //Declare the variable for digital io LED1.
  DigitalIn mysw(D10);      //Declare D10 pin as Digital Input Pin

  int main() {

    int sw_val;

    while(1){ //Execute forever.

      sw_val = mysw.read();    //Read Digital input pin

      printf("Digital Input value is %d \n\r", sw_val); //Debug Digital value

      if(sw_val == 1) //Check the value
      {     
        myled_R = 0;  //Making digital io LED1 to 0
      }
      else
      {
        myled_R = 1;  //Making digital io LED1 to 1
      }

      wait(0.2);
    }
  }

Digital Out

This Digital Out Example program is for control Digital Output Pin value. User is able to control LED1 , LED2 , LED3 with Ditigal Output Pin value.

Importing Digital Out

This is the Digial Out Example program. Click the Import into Compiler button



Let’s open the main.cpp file in Program Workspace. This program declares LED1 , LED2 , LED3 as Digital Output and controls Digital Output Value . 

#include "mbed.h"

DigitalOut myled_R(LED1); //Declare the variable for digital io LED1.
DigitalOut myled_G(LED2); //Declare the variable for digital io LED2.
DigitalOut myled_B(LED3); //Declare the variable for digital io LED3.

int main()
{    
    while(1) {  //Execute forever.
        myled_R = 0;  //Making digital io LED1 to 0
        myled_G = 0;  //Making digital io LED1 to 0
        myled_B = 0;  //Making digital io LED1 to 0
        wait(0.5);  //Delay 0.5sec
        myled_R = 1;  //Making digital io LED1 to 1
        myled_G = 1;  //Making digital io LED1 to 1
        myled_B = 1;  //Making digital io LED1 to 1
        wait(0.5);  //Delay 0.5sec
    }
}
