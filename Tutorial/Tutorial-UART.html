UART

This UART example is that UART communication between PC to WIZwiki-W7500 . When user type a character on Serial Terminal , PC sends it to WIZwiki-W7500 via USB cable and WIZwiki-W7500 returns the character + 1 . Then, it is monitered on Serial Terminal.

Importing UART Example

This is the UART Example program. Enter the Import into Compiler button.



The Mbed library supports the following class references for UART . For more information, see the Link below.

Mbed library link



Let’s open the main.cpp file in Program Workspace. This program used Serial class from mbed library for UART . This program using UART1 on WIZwiki-W7500 . UART1 TXD , UART1 RXD pins are PC_2 and PC_3 . Baudrate is set 115200bps . 

#include "mbed.h"

//Generate pc Object from Serial Library
//Serial(/*TXD*/,/*RXD*/);
Serial pc(PC_02,PC_03);

int main(){

    pc.baud(115200); //Call baud() to set Baudrate

    pc.printf("Hello World!\n");

    while(1)
    {        
        pc.putc(pc.getc()+1); //Output the input value + '1'
    }
}

Result

Serial Terminal input (Hercules)

  1. Download firmware into Wizwiki W7500.
  2. Open Serial Terminal in PC.
  3. Make sure that Port name and 9600bps(Baudrate) and connect WIZwiki-W7500 .
  4. When firmware starts, Serial Terminal shows Hello World! .
  5. Send a character to WIZwiki-W7500.
  6. WIZwiki-W7500 will return the character value + 1
